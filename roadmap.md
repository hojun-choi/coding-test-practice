# 8-Week Roadmap

## Week 1

- Topic: Array / String / Hash
- Day 1 Concept Goal: understand array and string basics, `dict`, `set`, frequency counting, duplicate check, and complement lookup
- Day 1 Concept Example: Two Sum
- Day 2~4 Target Problems:
  - Contains Duplicate
  - Valid Anagram
  - Longest Common Prefix
  - Best Time to Buy and Sell Stock
  - Group Anagrams
  - Top K Frequent Elements
  - Majority Element
  - Product of Array Except Self
  - Maximum Subarray
  - Two Sum retry
- Retry Target: Two Sum

## Week 2

- Topic: Two Pointers / Sliding Window / Prefix Sum
- Day 1 Concept Goal: understand two pointers, sliding window, prefix sum, and left/right movement rules
- Day 1 Concept Example: Valid Palindrome
- Day 2~4 Target Problems:
  - Move Zeroes
  - Two Sum II - Input Array Is Sorted
  - Container With Most Water
  - 3Sum
  - Longest Substring Without Repeating Characters
  - Maximum Average Subarray I
  - Find Pivot Index
  - Minimum Size Subarray Sum
  - Subarray Sum Equals K
  - Valid Palindrome retry
- Retry Target: Valid Palindrome

## Week 3

- Topic: Stack / Queue / Linked List
- Day 1 Concept Goal: understand stack, queue, deque, linked list, dummy node, and fast/slow pointer
- Day 1 Concept Example: Valid Parentheses
- Day 2~4 Target Problems:
  - Implement Queue using Stacks
  - Min Stack
  - Evaluate Reverse Polish Notation
  - Daily Temperatures
  - Reverse Linked List
  - Merge Two Sorted Lists
  - Linked List Cycle
  - Remove Nth Node From End of List
  - Add Two Numbers
  - Valid Parentheses retry
- Retry Target: Valid Parentheses

## Week 4

- Topic: BFS / DFS / Graph / Tree
- Day 1 Concept Goal: understand DFS recursion, BFS queue, visited set, grid traversal, tree traversal, and adjacency list
- Day 1 Concept Example: Flood Fill
- Day 2~4 Target Problems:
  - Maximum Depth of Binary Tree
  - Number of Islands
  - Max Area of Island
  - Rotting Oranges
  - Binary Tree Inorder Traversal
  - Same Tree
  - Invert Binary Tree
  - Course Schedule
  - Clone Graph
  - Flood Fill retry
- Retry Target: Flood Fill

## Week 5

- Topic: Binary Search / Sorting / Intervals
- Day 1 Concept Goal: understand binary search, sorted arrays, answer-space binary search, and interval merge
- Day 1 Concept Example: Binary Search
- Day 2~4 Target Problems:
  - Search Insert Position
  - Search in Rotated Sorted Array
  - Find Minimum in Rotated Sorted Array
  - First Bad Version
  - Find Peak Element
  - Koko Eating Bananas
  - Merge Intervals
  - Insert Interval
  - Non-overlapping Intervals
  - Binary Search retry
- Retry Target: Binary Search

## Week 6

- Topic: Heap / Greedy
- Day 1 Concept Goal: understand heap, priority queue, Top K patterns, greedy decisions, and sorting plus greedy
- Day 1 Concept Example: Last Stone Weight
- Day 2~4 Target Problems:
  - Kth Largest Element in an Array
  - Top K Frequent Elements
  - K Closest Points to Origin
  - Assign Cookies
  - Jump Game
  - Gas Station
  - Partition Labels
  - Task Scheduler
  - Hand of Straights
  - Last Stone Weight retry
- Retry Target: Last Stone Weight

## Week 7

- Topic: DP
- Day 1 Concept Goal: understand DP state, recurrence, base case, memoization, tabulation, 1D DP, and 2D DP
- Day 1 Concept Example: Climbing Stairs
- Day 2~4 Target Problems:
  - Min Cost Climbing Stairs
  - House Robber
  - House Robber II
  - Fibonacci Number
  - Coin Change
  - Partition Equal Subset Sum
  - Longest Increasing Subsequence
  - Unique Paths
  - Longest Common Subsequence
  - Climbing Stairs retry
- Retry Target: Climbing Stairs

## Week 8

- Topic: Matrix / Implementation / Design
- Day 1 Concept Goal: understand matrix traversal, simulation, in-place update, backtracking, and basic design problems
- Day 1 Concept Example: Valid Sudoku
- Day 2~4 Target Problems:
  - Spiral Matrix
  - Set Matrix Zeroes
  - Rotate Image
  - Word Search
  - Pacific Atlantic Water Flow
  - Surrounded Regions
  - Reorder List
  - LRU Cache
  - Design Browser History
  - Valid Sudoku retry
- Retry Target: Valid Sudoku
