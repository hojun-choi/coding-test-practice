# Dynamic Programming

## Short Explanation

DP는 큰 문제를 작은 부분 문제로 나누고, 이미 계산한 결과를 재사용하는 방식이다. 먼저 `상태`, `점화식`, `초기값`을 정리해야 한다.

## When To Use

- 같은 계산이 반복될 때
- 선택지가 누적되어 최적값/경우의 수를 구할 때
- 앞의 결과가 뒤의 의사결정에 영향을 줄 때

## Basic Python Template

```python
def fib(n):
    if n <= 1:
        return n

    dp = [0] * (n + 1)
    dp[1] = 1

    for i in range(2, n + 1):
        dp[i] = dp[i - 1] + dp[i - 2]

    return dp[n]
```

## Common Mistakes

- 상태 정의 없이 점화식을 억지로 외움
- 초기값을 빠뜨려 인덱스 에러가 남
- 가능 여부 문제와 최소/최대 문제를 같은 방식으로 접근함
- 공간 최적화가 가능한데 배열을 끝까지 유지함
